문서편집기 vi 
1) 파일 만들기 vi 파일명

1-2) 텍스트 입력하기 
i 클릭하면 하단에 --INSERT-- 라고 나오면 원하는 글씨 입력

2) 저장하기 
(--INSERT-- 상태라면 esc 클릭후)
:w -> 엔터
3) 저장하지 않고 나오기
(--INSERT-- 상태라면 esc 클릭후)
:q! -> 엔터

4) 저장하고 즉시 나오기
(--INSERT-- 상태라면 esc 클릭후)
:wq -> 엔터 

5) 문자 검색
/검색문자 -> 엔터
5-2) 아래로 내려가면서 계속 찾기
/검색문자 -> 엔터 누른 상태에서 n 버튼 클릭 (반대 방향은 N (쉬프트+n) 버튼 클릭) 

6) 특정 문자 일괄 바꾸기
(--INSERT-- 상태라면 esc 클릭후)
:%s/타겟문자/바꿀문자
